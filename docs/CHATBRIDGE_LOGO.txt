                            WHAT IS CHATBRIDGE?

   ChatBridge is an enterprise-grade AI platform that bridges multiple
   LLM providers through a unified, secure, beautiful chat interface.

   ✨ Features:
   • Multiple LLM Providers (OpenAI, Anthropic, Azure, etc.)
   • Unified Chat Interface (Open WebUI)
   • Intelligent Routing (LiteLLM)
   • Enterprise Security
   • Auto SSL/TLS
   • Daily Backups
   • One-Command Installation
   • Production Ready
