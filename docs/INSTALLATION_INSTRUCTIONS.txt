🚀 INSTALLATION IN 3 STEPS:

STEP 1: Download and Extract
────────────────────────────

   $ wget [your-url]/chatbridge-stack.tar.gz
   $ tar -xzf chatbridge-stack.tar.gz
   $ cd chatbridge-stack/


STEP 2: Run Setup
─────────────────────────────

   $ chmod +x setup.sh
   $ ./setup.sh

   Provide when prompted:
   - Your domain name
   - Your email (for SSL)
   - API keys


STEP 3: Access ChatBridge
─────────────────────────

   Wait 2-5 minutes, then visit:
   
   🌉 https://your-domain.com

   Create your admin account and start bridging!
