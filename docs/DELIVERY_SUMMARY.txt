ChatBridge is an enterprise-grade AI platform that bridges multiple LLM
providers (OpenAI, Anthropic, Azure, etc.) through a unified, beautiful
chat interface - with production-ready security, automated operations,
and zero-touch deployment.

**The Perfect Bridge for Small Business AI**

📋 INSTALLATION CHECKLIST
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


BEFORE INSTALLATION:
□ Linux server (Ubuntu 20.04+)
□ 4GB+ RAM (8GB recommended)
□ 20GB+ disk space
□ Domain name registered
□ DNS configured (A records)
□ Ports 80/443 accessible
□ API keys ready (OpenAI, Anthropic, etc.)


INSTALLATION:
□ Extract files to /opt/chatbridge
□ Run: ./setup.sh
□ Provide domain, email, API keys
□ Wait 5-10 minutes


AFTER INSTALLATION:
□ Visit https://your-domain.com
□ Create admin account
□ Disable signup: ENABLE_SIGNUP=false
□ Run: ./health-check.sh
□ Configure firewall
□ Test backup: ./manage.sh
